# Question 1

Given an array of strings (`"<clicks>,<subdomain>.<subdomain>.<tld>"`), find the total number of clicks for each domain (e.g. `<subdomain>`, `<subdomain>.<tld>`, etc.).

**Example**:

```
[
  "900,google.com",
  "60,mail.yahoo.com",
  "10,mobile.sports.yahoo.com",
  "40,sports.yahoo.com",
  "300,yahoo.com",
  "10,stackoverflow.com",
  "20,overflow.com",
  "5,com.com",
  "2,en.wikipedia.org",
  "1,m.wikipedia.org",
  "1,mobile.sports",
  "1,google.co.uk"
]
```
